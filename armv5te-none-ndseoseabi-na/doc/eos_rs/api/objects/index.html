Module eos_rs::api::objects

source · [−]
Expand description

Type aliases and small wrapper types for common ffi types and modules. These are generally to be preferred over directly using the ffi types and modules.

The ..._catalog re-exports are catalogs of IDs of items, moves, etc. that exist in the base game. The Type item of these catalogs specifies the type of the IDs.
