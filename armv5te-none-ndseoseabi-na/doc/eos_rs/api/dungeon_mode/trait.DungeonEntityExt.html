Expand description

Extension trait for DungeonEntity.

Important safety note

Implementations of this trait can assume that overlay 29 is loaded (since this is the only context dungeon entities are actually relevant). If you manually implement this trait, for some reason, you NEED to make sure overlay 29 is loaded when using some functions of this trait. If you use the DungeonEntity structs manually outside of dungeon mode, this trait will be unsafe to use. The trait and its functions are marked safe for convenience, since in its intended use case, overlay 29 will always be loaded.
